# 🧦 SockStore – Full Stack E-commerce Application

A modern Full Stack e-commerce web application developed using **Angular**, **ASP.NET Core**, **C#**, **Entity Framework Core**, and **PostgreSQL**.

The application allows customers to browse products, manage a shopping cart and wishlist, place orders, and track their purchases. It also includes an administrative dashboard for managing products, inventory, and customer orders.

---

# 🚀 Features

### Customer Features

* User registration and login (JWT Authentication)
* Browse products
* Product details
* Shopping cart
* Wishlist
* Secure checkout
* Order history
* Responsive interface

### Administrator Features

* Dashboard
* Product management (Create, Edit, Delete)
* Inventory management
* Order management
* Update order status

# ⚙ Installation

## Clone the repository

```bash
git clone https://github.com/CodexUp/sockstore-fullstack.git
```

---

## Backend

```bash
cd backend

dotnet restore

dotnet run
```

---

## Frontend

```bash
cd frontend

npm install

ng serve
```
